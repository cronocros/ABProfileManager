# ABProfileManager v1.3.3

패치 릴리스입니다.

## 이번 버전 핵심

- 전문기술 탭 카드 레이아웃을 다시 정리했습니다.
  - 긴 source 이름과 진행 문구가 카드 오른쪽으로 넘치지 않도록 row 폭과 높이를 다시 조정했습니다.
  - 상단 제어 영역을 다시 배치해 `전문기술 포인트 오버레이 표시` 체크박스가 메인 UI 밖으로 벗어나지 않도록 수정했습니다.
- 전문기술 오버레이 크기 조절을 추가했습니다.
  - 전문기술 탭에서 `작게 / 기본 / 크게` 버튼으로 오버레이 크기를 바로 바꿀 수 있습니다.
  - 접기 버튼도 더 작은 아이콘형 버튼으로 축소했습니다.
- 스탯 오버레이 크기 조절을 추가했습니다.
  - 설정 탭에서 `작게 / 기본 / 크게` 버튼으로 전체 표시 크기를 바꿀 수 있습니다.
- 한밤(Midnight) 지도 오버레이 줌 곡선을 다시 조정했습니다.
  - 하란다르, 줄아만, 영원노래숲 같은 외부 지도에서 지도를 축소했을 때 던전 / 구렁 라벨이 너무 작아지지 않도록 최소 체감을 더 높였습니다.

## 참고

- 전문기술 / 스탯 오버레이 크기 조절은 프레임 전체 스케일을 기준으로 동작합니다.
- 지도 라벨은 여전히 정적 좌표 기반이라, 실제 게임 패치 후 좌표가 달라지면 미세 조정이 추가로 필요할 수 있습니다.
