# ABProfileManager v1.2.0

릴리스 날짜: `2026-03-12`

## 핵심 변경

- 전문기술 주간 체크를 숨은 퀘스트 기반 자동 추적 구조로 전환
- 주간 퀘스트, Treatise, 주간 드랍, 1회성 보물, 평판 지식서, 일부 풍요 지식서, 약초/채광 처음 발견 보너스 자동 계산 지원
- 전문기술 탭에 `재스캔` 버튼과 소스별 진행 툴팁 추가
- 설정 탭 레이아웃 정리로 체크박스 영역 넘침 수정
- Midnight 지도 오버레이 가독성 개선
- 실버문 포탈 이름, 던전/구렁 입구, 정복 상인, 교역소, 형상변환, 암시장 표시 추가
- Eversong Woods / Harandar / Voidstorm / Zul'Aman 지도에 평판 상인과 주요 입구 이름 표시 추가

- 지도 오버레이는 Midnight 주요 지도 기준의 정적 좌표를 사용합니다.
- 제작 주문, catch-up 같은 일부 예외 전문기술 획득원은 이번 버전 자동 집계 범위에 포함하지 않았습니다.
