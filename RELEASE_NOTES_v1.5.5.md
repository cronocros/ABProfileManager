# ABProfileManager v1.5.5

릴리스 날짜: 2026-03-30

## 요약

이번 릴리스는 인게임 QA 후속 이슈를 반영한 UI/툴팁 정리 릴리스입니다. BIS 오버레이의 시즌 미리보기 정확도를 다시 손봤고, 드랍템 레벨 오버레이 우측 패널과 기본 설정 오버플로우를 함께 정리했습니다.

## 주요 변경

### 1. BIS 인던 드랍 정보

- 오래된 던전이 예전 Encounter Journal 인스턴스로 고정 매칭되던 경로를 제거했습니다.
- 던전명 기준으로 최신 tier 후보를 수집한 뒤, 실제 시즌 preview loot link를 찾은 candidate를 우선 사용하도록 변경했습니다.
- 아이템 클릭 시 가능하면 `Encounter Journal -> Dungeon -> Loot` 화면까지 바로 열도록 조정했습니다.
- 헤더에 `스크롤로 크기조절` 힌트와 현재 아이템 레벨을 추가했습니다.
- `쐐기 / 레이드 / 제작` 소스 필터 버튼을 추가했습니다.
- 특성 아이콘, 드롭다운, 필터 버튼 정렬을 다시 잡았습니다.
- 아이템명 / 던전 / 우선순위 컬럼 폭을 조정하고 던전 컬럼 줄바꿈을 막았습니다.

### 2. 드랍템 레벨 정보

- 구렁 탭의 `?/?` 등급 표기를 제거하고 정확한 등급명만 표시하도록 변경했습니다.
- `종합 / 쐐기 / 구렁 / 레이드 / 기타` 탭 상단에 공통 요약 타이틀을 추가했습니다.
- `나의 문장` 패널 시작 위치를 더 아래로 내리고 글자 크기를 키웠습니다.
- 우측 패널에 `나의 열쇠` 섹션을 추가했습니다.
- `복원된 열쇠`는 Delves currency 기준으로 표시합니다.

### 3. 설정 UI

- Blizzard 기본 설정의 전투메시지 블록에서 설명문을 숨기고 모드 버튼 폭을 줄여 오버플로우를 완화했습니다.
- 메인 설정창 레이아웃은 그대로 유지했습니다.

## 확인 메모

- `열쇠 파편`은 Blizzard 공개 UI source 기준으로 안전한 itemID를 아직 확정하지 못해 `-` fallback이 남아 있습니다.
- `오늘의 풍요 구렁`은 공개 API에 단일 조회 함수가 없어 active delve / map POI 기준 best-effort 표시를 사용합니다.
