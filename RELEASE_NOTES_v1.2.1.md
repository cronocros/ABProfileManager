# ABProfileManager v1.2.1

릴리스 날짜: `2026-03-12`

## 핵심 변경

- 고스트 자동 재시도 큐가 커서 사용 중에는 멈추도록 바꿔, 고스트 위에 탈것이나 다른 액션을 덮어쓸 때 아이콘이 즉시 떨어지거나 사라지는 간헐 버그를 완화
- 스탯 오버레이 상단에 `캐릭터 - 직업 - 특성` 헤더를 추가하고, 프레임 폭과 힛트박스를 실제 텍스트 영역에 가깝게 축소
- 스탯 퍼센트 정수부에 투명 패딩을 넣어 한 자리수 / 두 자리수 / 세 자리수 정렬을 다시 보정
- 설정 탭 하단 설명 박스를 현재 캐릭터, 오버레이 상태, 전문기술 마지막 스캔, 디버그 세션 상태를 보여주는 요약 영역으로 교체
- 전문기술 자동 추적 결과 캐시와 Midnight 지도 오버레이 드라이버 상태 제어를 추가해 불필요한 반복 갱신 비용을 줄임

## 참고

- 고스트 덮어쓰기 버그는 자동 재시도와 수동 커서 드래그가 겹치는 경로를 우선 차단한 패치입니다.
- 스탯 오버레이와 Midnight 지도 오버레이의 최종 위치/가독성은 인게임에서 한 번 더 미세 확인하는 것이 좋습니다.
