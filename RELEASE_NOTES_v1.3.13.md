# ABProfileManager v1.3.13

배포일: 2026-03-13

## 핵심 변경

- TomTom 기반 미완료 전문기술 보물 waypoint 기능을 인트로와 사용자 문서의 주요 포인트로 정리
- 하란다르와 공허폭풍 일부 보물은 별도 지역 지도라서, 해당 지역에 들어가면 waypoint가 생성된다는 설명으로 통일
- profession 오버레이, 설정 세션 안내, 상태 메시지의 TomTom 문구를 실제 동작 기준으로 업데이트
- 문서 전반과 릴리스 메타데이터를 `v1.3.13` 기준으로 다시 최신화

## 사용자 영향

- TomTom이 설치되어 있으면 profession 오버레이의 `1회성` 줄 우클릭으로 미완료 보물 waypoint를 선택할 수 있습니다.
- 하란다르/공허폭풍 일부 보물은 다른 지역에서 바로 찍히지 않아도 버그가 아니라, 해당 지역에 들어가면 정상적으로 waypoint가 생성됩니다.
- 관련 설명은 인게임 안내, README, 사용자 문서, 인트로에 모두 반영되었습니다.
