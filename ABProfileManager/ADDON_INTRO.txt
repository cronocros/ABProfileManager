ABProfileManager

특성 전환 뒤 엉킨 액션바 복구, 매주 챙겨야 하는 전문기술 포인트 확인, 한밤(Midnight) 지역 길찾기를 한 번에 줄여주는 WoW Retail용 관리 애드온입니다.

버전: v1.3.5
제작: 밍밍이와코코
연락처: crono1232@gmail.com

[왜 쓰는가]
- 캐릭터별 액션바를 템플릿으로 저장하고, 필요한 범위만 빠르게 다시 적용할 수 있습니다.
- 전문기술 포인트는 자동 추적 카드와 오버레이로 매주 확인할 수 있습니다.
- 한밤(Midnight) 지도에서는 주요 시설, 포탈, 던전, 구렁 위치를 읽기 쉬운 라벨로 바로 찾을 수 있고, 카테고리별로 켜고 끌 수 있습니다.
- 퀘스트 정리, 스탯 오버레이, 미니맵 버튼까지 함께 제공해 반복 작업을 줄입니다.

[대표 기능]
1. 액션바 템플릿
- 저장 / 복제 / 삭제 / 비교 / 동기화 / 부분 적용
- 최근 1회 작업 되돌리기
- 비행 중 바 포함 저장과 적용

2. 전문기술 주간 체크
- 주간 / 1회성 획득원 자동 추적
- profession 아이콘, 포인트 합계, 재스캔 지원
- 상세 / 요약 / 최소 3단 보기와 5단계 크기 조절 지원

3. 한밤(Midnight) 지도 오버레이
- 실버문과 외부 지역 주요 시설, 포탈, 평판 상인, 던전 / 구렁 입구 표시
- 카테고리별 on/off 체크박스로 필요한 정보만 선택 표시
- `던전:` / `구렁:` / `공격대:` 접두어와 줄바꿈 규칙 적용
- 지도 확대 / 축소에 맞춰 라벨 크기를 자동 보정

4. 추가 편의 기능
- 퀘스트 후보 새로고침 / 안전 정리 / 전체 포기
- 캐릭터 직업 - 특성(아이템레벨) 스탯 오버레이
- 전문기술 / 스탯 오버레이 5단계 크기 조절
- 전투 중 대기열 처리와 고스트 슬롯 안내

[추천 대상]
- 특성 전환 후 액션바 복구를 자주 하는 플레이어
- 캐릭터별 전문기술 포인트를 매주 챙기는 플레이어
- 한밤(Midnight) 지역 길찾기를 빠르게 하고 싶은 플레이어
- 여러 캐릭터를 돌리면서 반복 점검 시간을 줄이고 싶은 플레이어
