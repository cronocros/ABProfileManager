ABProfileManager
────────────────────────────────────────────────────

특성 바꿀 때마다 액션바가 엉망이 되시나요?
전문기술 주간 포인트, 이번 주에 다 챙겼는지 매번 까먹으시나요?
한밤(Midnight) 시즌 1 BIS 추천 장비를 게임 안에서 바로 보고 싶으신가요?

ABProfileManager는 그 작업들을 한 창과 몇 개의 오버레이로 정리합니다.

버전: v1.7.4
제작: 밍밍이와코코 (crono1232@gmail.com)

════════════════════════════════════════════════════
  핵심 기능
════════════════════════════════════════════════════

▶ 액션바 템플릿 관리
  특성별·캐릭터별 액션바 배치를 템플릿으로 저장하고,
  한 번의 클릭으로 정확히 복구합니다.
  - 저장 / 복제 / 비교 / 부분 적용 / 되돌리기 지원
  - 가져오기·내보내기로 다른 캐릭터나 지인과 공유 가능
  - 적용 전 확인 창으로 실수 방지

▶ 전문기술 포인트 자동 추적
  이번 주 주간 퀘스트, 논문, 드랍, 평판/풍요 지식서,
  1회성 보물까지 자동으로 계산합니다.
  - 완료·미완료 색상 구분과 포인트 진행도를 한눈에 확인
  - 화면 위 전문기술 오버레이로 창 열지 않고 빠르게 체크
  - TomTom 연동 시 미완료 보물을 오버레이 우클릭으로
    바로 waypoint 등록

▶ 한밤(Midnight) 지도 오버레이
  실버문, 영원노래 숲, 하란다르, 보이드스톰의
  포탈·시설·전문기술·평판상인·던전·구렁 위치를
  월드맵 위 텍스트 라벨로 표시합니다.
  - 필터로 원하는 종류만 골라 표시
  - 글자 크기를 슬라이더로 자유롭게 조절

▶ BIS 추천 장비 오버레이
  현재 캐릭터 특성 기준 한밤 시즌 1 추천 장비를
  부위별로 정리해 보여줍니다.
  - `쐐기 / 레이드 / 제작 / 티어` 4개 필터
  - 필터 후 살아남은 후보 전체를 부위별로 표시
  - 첫 2개는 `1순위 / 2순위`, 나머지는 `3순위+` 재번호화
  - `레이드 off + 쐐기만 on`이어도 쐐기 드랍템과 인던 유지
  - `mythicplus / raid`는 가능한 경우 모험 안내서 랜딩
  - `crafted / tier`는 랜딩하지 않음
  - 쐐기 항목에 던전 종료 / 위대한 금고 보상 트랙과 템렙 안내
  - 아이템 hover는 안전한 전용 툴팁으로 표시

▶ 드랍템 레벨 오버레이
  쐐기, 구렁, 레이드, 제작 기준의 드랍 아이템 레벨과
  위대한 금고 보상, 드랍 문장 등급을 한 표로 확인합니다.
  - `단/난이도 | 클리어보상 | 드랍문장 | 위대한 금고` 4열 구조
  - 우측 고정 `나의 문장` 패널로 현재 보유량 통합 표시
  - `나의 열쇠` 패널에서 오늘의 풍요 4개와 열쇠 자원 확인
  - 쐐기 타이틀 옆에 챔피언 / 영웅 / 신화 최고 강화 레벨 요약
  - 마우스 휠로 오버레이 크기 조절

▶ 파티찾기 시즌 최고기록 아이콘 오버레이
  신화+ 던전 탭의 시즌 최고기록 던전 이미지 위에
  `평점 + 던전명`을 직접 표시합니다.
  - Utility 탭 체크박스로 켜고 끌 수 있습니다

▶ 캐릭터 스탯 오버레이
  치명/가속/특화/유연 수치와 현재 특성의 선호 스탯
  우선순위를 화면 어디든 떠 있는 오버레이로 확인합니다.
  - 탱커 특성일 때 회피·반격·막기 수치 함께 표시
  - 수치에 마우스를 올리면 전용 안전 툴팁으로 설명 표시
  - 메인 창 `스탯 우선순위 표`에서 40개 전문화 전체 확인

▶ 전투메시지 모드 설정
  한밤(Midnight) 클라이언트에서 기본 UI에 노출되지 않는
  전투메시지 표출 방식을 애드온 설정 탭에서 바로 적용합니다.
  - 위로 / 아래로 / 부채꼴(아크) 모드 선택 및 즉시 적용
  - 로그인·월드 진입 직후 선택한 모드를 자동 재적용
  - 방향성 분산 on/off

════════════════════════════════════════════════════
  v1.7.4 업데이트
════════════════════════════════════════════════════

  - 액션바 / 모험 안내서 / Pawn 아이템 hover에서 발생할 수 있던
    `MoneyFrame.lua secret number` 오류를 막았습니다.
  - WoW 12.0.7 계열 클라이언트 대응을 위해 Interface 번호에 120007을 추가했습니다.
  - ABPM 자체 hover 설명은 전역 GameTooltip 대신 애드온 전용 툴팁을
    사용하도록 바꿨습니다.
  - BIS 아이템 hover는 Blizzard 기본 툴팁을 직접 열지 않고,
    판매가 라인을 제외한 안전한 텍스트 툴팁으로 표시합니다.
  - 쐐기 BIS 항목에 던전 종료 / 위대한 금고 보상 트랙과
    대표 아이템 레벨 안내를 추가했습니다.
  - Patch 12.0.5 기준 40개 전문화 `스탯 우선순위 표`를 추가했습니다.
  - 첫 설치 언어는 WoW 클라이언트 기준으로 적용됩니다.
    한국어 클라이언트는 한국어, 영어/미지원 클라이언트는 영어로 열립니다.
  - 이전 버전에서 영어 클라이언트에 한국어가 저장된 경우 한 번만 영어로
    자동 보정합니다.
  - 한밤 시즌 1 BIS 카탈로그와 한글/영문 출처 표기를 다시 정리했습니다.

════════════════════════════════════════════════════
  Midnight 버그 자동 수정
════════════════════════════════════════════════════

▶ 마우스 이동 자동 복구
  한밤(Midnight)에서 NPC·오브젝트 상호작용 후
  마우스 이동(우클릭 드래그)이 꺼지는 버그가 있습니다.
  설정 탭에서 "마우스 이동 자동 복구"를 켜두면
  로그인 시 자동으로 다시 활성화해 이 버그를 방지합니다.

════════════════════════════════════════════════════
  이런 분께 추천합니다
════════════════════════════════════════════════════

  - 특성 전환이나 부캐 플레이가 잦아 액션바 정리가 번거로운 분
  - 전문기술 포인트를 매주 빠짐없이 챙기고 싶은 분
  - 한밤(Midnight) 지역 길찾기에 시간을 낭비하고 싶지 않은 분
  - BIS와 드랍 템렙을 게임 안에서 바로 확인하고 싶은 분
  - 내 캐릭터 스탯을 플레이 중에도 한눈에 확인하고 싶은 분
  - 부채꼴 모드가 로그인마다 풀려서 스트레스 받으신 분
