ABProfileManager

액션바가 꼬이면 바로 복구하고, 전문기술 포인트는 놓치지 않고, 한밤(Midnight) 지역 길찾기는 헤매지 않게 만들어 주는 WoW Retail 애드온입니다.

버전: v1.3.16
제작: 밍밍이와코코
연락처: crono1232@gmail.com

[이 애드온이 하는 일]
- 특성 전환이나 캐릭터 변경 뒤 엉킨 액션바를 저장한 템플릿으로 빠르게 복구합니다.
- 전문기술 포인트를 주간과 1회성으로 나눠 자동 추적해, 이번 주에 무엇을 놓쳤는지 바로 확인할 수 있습니다.
- TomTom이 있으면 미완료 1회성 전문기술 보물 waypoint를 바로 찍을 수 있습니다.
- 한밤(Midnight) 지역의 포탈, 허브, 던전, 구렁, 공격대 입구를 지도 오버레이로 바로 보여줍니다.
- 전투메시지 옵션이 기본 게임 설정에 잘 보이지 않는 환경에서도, 설정 탭에서 직접 원하는 방식으로 바꿀 수 있습니다.

[왜 쓰면 좋은가]
- 여러 캐릭터를 돌릴 때 액션바를 다시 맞추는 시간을 크게 줄일 수 있습니다.
- 전문기술 포인트를 매주 기억에 의존하지 않고, 화면에서 바로 점검할 수 있습니다.
- 길찾기, 체크리스트, 액션바 복구를 각각 다른 애드온이나 메모로 관리하지 않아도 됩니다.
- 메인 창 하나와 몇 개의 오버레이로 반복 작업을 줄이는 데 초점을 맞췄습니다.

[이런 플레이어에게 추천]
- 특성 전환이 잦아서 액션바가 자주 꼬이는 플레이어
- 여러 캐릭터의 전문기술 포인트를 주간 단위로 챙기는 플레이어
- 한밤(Midnight) 지역 포탈, 전문기술 허브, 던전/구렁 위치를 빠르게 찾고 싶은 플레이어
- 게임 옵션에 없는 전투메시지 배치를 직접 조절하고 싶은 플레이어

[대표 기능]
1. 액션바 템플릿 관리
- 저장 / 비교 / 부분 적용 / 동기화 / 최근 1회 되돌리기

2. 전문기술 자동 추적
- 주간 포인트와 1회성 보물 진행도 자동 계산
- 상세 / 요약 / 최소 오버레이
- TomTom waypoint 연동

3. 한밤(Midnight) 지도 오버레이
- 주요 시설, 포탈, 전문기술 허브, 던전, 구렁, 공격대 입구 표시

4. 전투메시지 설정 복구
- 위로 / 아래로 / 부채꼴 모드 선택
- 피해 숫자, 치유 숫자, 방향성 분산 표시 제어
