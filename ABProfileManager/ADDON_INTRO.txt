ABProfileManager

액션바가 꼬일 때는 빠르게 복구하고, 전문기술 포인트는 놓치지 않고, 한밤(Midnight) 지역 길찾기는 지도에서 바로 끝내고 싶은 플레이어를 위한 WoW Retail 애드온입니다.

버전: v1.3.16
제작: 밍밍이와코코
연락처: crono1232@gmail.com

[왜 써야 하나]
- 특성 전환 뒤 엉킨 액션바를 템플릿으로 바로 복구할 수 있습니다.
- 전문기술 포인트를 캐릭터별로 자동 추적해 매주 체크 시간을 줄여줍니다.
- TomTom 설치 시 미완료 1회성 전문기술 보물 waypoint를 바로 찍을 수 있습니다.
- 한밤(Midnight) 지역의 포탈, 주요 시설, 던전, 구렁, 공격대 입구를 지도 위 글자로 바로 찾을 수 있습니다.
- 퀘스트 정리, 스탯 오버레이, 미니맵 버튼까지 함께 제공해 반복 관리 시간을 줄여줍니다.

[대표 포인트]
1. 액션바 템플릿 관리
- 저장 / 복제 / 삭제 / 비교 / 동기화 / 부분 적용
- 최근 1회 작업 되돌리기
- 비행 중 바 포함 저장과 적용

2. 전문기술 포인트 자동 추적
- 주간 / 1회성 획득원 자동 추적
- profession 카드와 profession 오버레이 동시 제공
- TomTom 설치 시 미완료 1회성 보물 waypoint 지원
- 상세 / 요약 / 최소 3단 보기
- 5단계 크기 조절 지원
- 오버레이 상단 요약 포인트 표기 통일과 tooltip 가독성 보강
- 전투메시지 CVar를 설정 탭에서 직접 제어하고 로그인 시 다시 적용

3. 한밤(Midnight) 지도 오버레이
- 실버문과 외부 지역 주요 위치를 읽기 쉬운 글자로 표시
- 카테고리별 필터로 필요한 정보만 표시
- 줌 보정, 줄바꿈 예외, 충돌 완화 적용

4. 추가 편의 기능
- 퀘스트 안전 정리 / 전체 포기 / 후보 목록
- 캐릭터 스탯 오버레이
- 전투 중 대기열과 고스트 안내

[추천 대상]
- 특성 전환이 잦은 플레이어
- 여러 캐릭터의 전문기술 포인트를 매주 챙기는 플레이어
- 한밤(Midnight) 지역 길찾기를 빠르게 끝내고 싶은 플레이어
