ABProfileManager

반복되는 액션바 정리와 전문기술 체크를 한 번에 끝내는 WoW Retail용 관리 애드온입니다.

버전: v1.3.2
제작: 밍밍이와코코

[한눈에 보는 핵심]
- 액션바를 템플릿으로 저장하고, 필요할 때 전체 또는 일부 범위만 다시 적용
- 비교, 동기화, 최근 1회 되돌리기로 세팅 복구를 더 빠르게 처리
- 전문기술 주간 / 1회성 포인트를 자동 추적하고 오버레이로 바로 확인
- 한밤(Midnight) 지도에서 주요 시설, 포탈, 던전, 구렁 위치를 읽기 쉬운 글자로 표시
- 퀘스트 정리, 스탯 오버레이, 미니맵 버튼까지 한 애드온 안에서 제공

[이런 플레이어에게 맞습니다]
- 캐릭터별 액션바를 자주 바꾸는 사람
- 특성 전환 후 액션바 복구가 번거로운 사람
- 전문기술 포인트 획득 현황을 매주 챙기고 싶은 사람
- 한밤(Midnight) 도시와 필드에서 주요 위치를 빠르게 찾고 싶은 사람

[대표 기능]
1. 액션바 템플릿 관리
- 저장 / 복제 / 삭제 / 내보내기 / 가져오기
- 전체 적용 / 부분 적용 / 비교 / 동기화
- 최근 1회 작업 되돌리기
- 비행 중 바 포함 저장과 적용

2. 전문기술 주간 체크
- profession별 주간 / 1회성 획득원 자동 추적
- profession 아이콘, 포인트 합계, 재스캔 지원
- 3단 접기 상태를 지원하는 profession 오버레이 제공

3. 한밤(Midnight) 지도 오버레이
- 주요 시설, 포탈, 정복 상인, 평판 상인, 전문기술 허브 표시
- 던전 / 구렁 입구 이름 표시
- 지도 확대 / 축소에 맞춰 글자 크기를 완만하게 조정

4. 추가 편의 기능
- 퀘스트 후보 새로고침 / 안전 정리 / 전체 포기
- 캐릭터 직업 - 특성(아이템레벨) 스탯 오버레이
- 전투 중 대기열 처리와 고스트 슬롯 안내

[상세 기능]
- 고스트 슬롯은 드래그 해제나 다른 액션 덮어쓰기를 지원합니다.
- 전문기술 오버레이는 화면 이동, 요약형, 최소형 표시를 지원합니다.
- 메인 창은 일반 WoW 창보다 앞에서 보이도록 보강되어 작업 중 묻히지 않습니다.
- 와우 `설정 > 애드온` 아래에 템플릿 / 액션바 / 전문기술 / 퀘스트 하위 카테고리가 추가됩니다.
