ABProfileManager는 월드 오브 워크래프트 Retail용 액션바 템플릿 애드온입니다.

버전: v1.3.0
상태: 1차 출시 완료

현재 캐릭터의 액션바 배치를 템플릿으로 저장하고,
필요할 때 전체 또는 일부 범위만 다시 적용할 수 있습니다.

주요 기능:
- 템플릿 저장 / 복제 / 삭제 / 문자열 가져오기 / 내보내기
- 전체 적용 / 부분 적용 / 비교 / 동기화 / 최근 1회 되돌리기
- 비행 중 바를 포함한 템플릿 처리
- 전투 중 대기열 처리와 고스트 오버레이
- 퀘스트 정리와 전체 퀘스트 포기
- 캐릭터 스탯 오버레이
  - 캐릭터 직업 - 특성(아이템레벨) 헤더
  - 치명 / 가속 / 특화 / 유연 `평점 (퍼센트)` 표시
  - 특성별 일반 PvE 우선순위 표시
  - 탱커 특성 방어 스탯 추가
- 전문기술 주간 체크
  - profession별 주간 / 1회성 획득원 자동 추적
  - profession 아이콘, 포인트 합계, 재스캔 지원
  - 접기 / 펼치기 가능한 profession 오버레이 제공
- Midnight 지도 오버레이
  - 실버문, Eversong, Harandar, Voidstorm, Zul'Aman 지도 주요 시설 표시
  - 포탈 이름, 평판 상인, 전문기술 허브, 던전 / 구렁 입구 한글 라벨 표시
- 와우 `설정 > 애드온` 하위 카테고리 제공

기본 UI는 한국어이며, 제작자는 밍밍이와코코입니다.
