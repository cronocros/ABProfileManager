ABProfileManager는 월드 오브 워크래프트 Retail용 액션바 템플릿 애드온입니다.

버전: v1.0.4
상태: 1차 출시 완료

현재 캐릭터의 액션바 배치를 템플릿으로 저장하고,
필요할 때 전체 또는 일부 범위만 다시 적용할 수 있습니다.

주요 기능:
- 템플릿 저장 / 복제 / 삭제
- 같은 이름 저장 시 덮어쓰기 확인
- 최근 1회 작업 되돌리기
- 현재 액션바와 템플릿 비교
- 빈 칸만 채우기, 다른 칸만 맞추기, 템플릿에 없는 칸 비우기
- 현재 적용 가능한 칸만 맞추기
- 문자열 내보내기 / 가져오기
- 현재 특성 바로 전환
- 비행 중 바를 포함한 템플릿 처리
- 퀘스트 정리와 전체 퀘스트 포기
- 버전 표시, 확장된 템플릿 정보, 미니맵 버튼 제공
- 캐릭터 스탯 오버레이 제공
  - 치명 / 가속 / 특화 / 유연 `평점(퍼센트)` 표시
  - 현재 특성의 PvE 일반 스탯 우선순위 표시
  - 탱커 특성 방어 스탯 추가 표시
  - 마우스 오버 툴팁과 DR 단계별 숫자 색상 표시

기본 UI는 한국어이며, 제작자는 밍밍이와코코입니다.
