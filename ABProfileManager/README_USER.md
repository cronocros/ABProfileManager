# ABProfileManager 사용자 안내

## 애드온 소개

`ABProfileManager`는 월드 오브 워크래프트 Retail용 액션바 템플릿 관리 애드온입니다.

현재 캐릭터의 액션바를 템플릿으로 저장하고, 필요할 때 전체 또는 일부 범위만 다시 적용할 수 있습니다.

버전: `v1.0.2`
상태: 1차 출시 완료

제작: `밍밍이와코코`

공식 저장소:
- `https://github.com/cronocros/ABProfileManager`

## 주요 기능

- 액션바 배치를 템플릿으로 저장
- 저장한 템플릿 복제, 삭제
- 최근 1회 작업 되돌리기
- 전체 액션바 또는 일부 범위만 적용
- 현재 액션바와 템플릿 비교
- 비교 결과 전체를 스크롤로 확인
- 빈 칸만 채우기 / 다른 칸만 맞추기 / 템플릿에 없는 칸 비우기
- 문자열 내보내기 / 가져오기
- 현재 특성 바로 전환
- 비행 중 바를 `9번 바`로 함께 저장하고 적용
- 퀘스트 후보 새로고침 / 안전 정리 / 전체 퀘스트 포기

## 기본 사용법

1. 게임 안에서 `/abpm` 입력
2. `현재 접속 캐릭터` 탭에서 템플릿 이름 입력 후 `템플릿 저장`
3. 저장된 템플릿 선택
4. `액션바` 탭에서 적용 범위 선택
5. 필요하면 먼저 비교
6. 적용 또는 동기화 실행

## 템플릿 탭

- 현재 캐릭터 정보와 직업/특성 아이콘 표시
- 현재 특성 전환 버튼 제공
- 템플릿 목록에서 선택 가능
- `이동` 버튼으로 이전 / 다음 템플릿 선택 가능
- 같은 이름으로 저장하면 덮어쓰기 확인창이 먼저 표시됩니다.
- 템플릿 작업 영역에서 아래 기능 사용 가능
  - 선택한 템플릿 적용
  - 전체 액션바 비우기
  - 직전 작업 되돌리기
  - 템플릿 복제
  - 문자열 내보내기
  - 문자열 가져오기
  - 템플릿 삭제

## 액션바 탭

적용 범위는 아래 방식으로 선택할 수 있습니다.

- 전체
- 바 1개
- 여러 바
- 선택 바
- 칸 번호

동기화 작업은 아래 기능을 제공합니다.

- 비어 있는 칸만 채우기
- 템플릿에 없는 칸 비우기
- 다른 칸만 바꾸기
- 선택 범위를 그대로 맞추기
- 적용 가능한 칸만 맞추기
  - 템플릿에는 있지만 현재 캐릭터가 바로 쓸 수 없는 스킬/매크로/아이템은 건너뜁니다.
  - 그런 칸은 고스트를 새로 만들지 않고 현재 칸을 유지합니다.

액션바 탭에서도 `직전 작업 되돌리기` 버튼으로 최근 1회 작업을 복구할 수 있습니다.
동기화 버튼 위에 마우스를 올리면 설명이 나오고, 버튼을 누르면 현재 선택 범위를 기준으로 자세한 설명이 하단 요약창에 표시됩니다.

## 퀘스트 탭

- `퀘스트 후보 새로고침`
  - 현재 퀘스트 목록을 다시 읽어 정리 후보를 새로 계산합니다.
- `안전 정리 실행`
  - 포기 가능
  - 완료 아님
  - 보고 가능 아님
  - 진행도 없음
  인 퀘스트만 정리합니다.
- `전체 퀘스트 포기`
  - 현재 퀘스트 목록에서 포기 가능한 퀘스트를 모두 포기합니다.
  - 진행 중인 퀘스트도 포함될 수 있습니다.
- 숨김 퀘스트, 작업형 퀘스트, 현상금 계열 항목은 후보 목록에서 제외합니다.

## 주의 사항

- 전투 중에는 보호된 액션바를 바로 변경할 수 없습니다.
- 전투 중 실행한 작업은 대기열에 들어가고, 전투가 끝나면 자동으로 다시 시도합니다.
- 없는 주문, 아이템, 매크로는 고스트 표시로 남을 수 있습니다.
- 고스트 표시가 남은 칸은 아이콘을 끌어 빼서 해제할 수 있습니다.
- 고스트 표시 위에 다른 스킬, 아이템, 매크로를 올려 덮어쓸 수도 있습니다.
- 매크로는 이름과 본문을 기준으로 더 엄격하게 확인합니다.
- 되돌리기는 현재 접속 중 가장 최근 1회 작업만 복구할 수 있습니다.

## 미니맵 버튼

- 왼쪽 클릭: 애드온 창 열기 / 닫기
- 오른쪽 드래그: 버튼 위치 이동
- 둥근 아이콘형 버튼으로 표시됩니다.

## 명령어

- `/abpm`
- `/abpm help`
- `/abpm list`
- `/abpm savetemplate 템플릿이름`
- `/abpm apply template 템플릿이름`
- `/abpm delete template 템플릿이름`
- `/abpm undo`
- `/abpm debug on`
- `/abpm debug off`
