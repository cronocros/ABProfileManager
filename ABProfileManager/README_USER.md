# ABProfileManager 사용자 안내

## 애드온 소개

`ABProfileManager`는 월드 오브 워크래프트 Retail용 액션바 템플릿 관리 애드온입니다.

현재 캐릭터의 액션바를 템플릿으로 저장하고, 필요할 때 전체 또는 일부 범위만 다시 적용할 수 있습니다.

버전: `v1.3.10`
상태: 1차 출시 완료

제작: `밍밍이와코코`
연락처: `crono1232@gmail.com`

공식 저장소:
- `https://github.com/cronocros/ABProfileManager`

최신 다운로드:
- 릴리스 페이지: `https://github.com/cronocros/ABProfileManager/releases/latest`
- 직접 다운로드: `https://github.com/cronocros/ABProfileManager/releases/download/v1.3.10/ABProfileManager-v1.3.10.zip`

## 한눈에 보는 핵심

- 자주 바뀌는 액션바를 템플릿으로 저장하고, 필요한 범위만 다시 적용할 수 있습니다.
- 비교, 동기화, 전체 비우기, 최근 1회 되돌리기로 세팅 복구가 빠릅니다.
- 전문기술 포인트는 자동 추적 카드와 오버레이로 한 번에 확인할 수 있습니다.
- 한밤(Midnight) 지도 주요 시설과 던전 / 구렁 / 공격대 입구는 지도 위 글자 오버레이로 바로 찾을 수 있습니다.
- 지도 라벨은 카테고리별 체크박스로 필요한 종류만 골라서 볼 수 있습니다.
- 메인 UI는 앞에, 스탯/전문기술 오버레이는 기본 WoW 창을 가리지 않도록 뒤에 배치됩니다.
- 퀘스트 정리와 스탯 오버레이까지 함께 제공해 반복 작업을 줄여줍니다.

## 주요 기능

- 액션바 배치를 템플릿으로 저장
- 저장한 템플릿 복제, 삭제
- 같은 이름 저장 시 덮어쓰기 확인
- 최근 1회 작업 되돌리기
- 전체 액션바 또는 일부 범위만 적용
- 현재 액션바와 템플릿 비교
- 비교 결과 전체를 스크롤로 확인
- 빈 칸만 채우기 / 다른 칸만 맞추기 / 템플릿에 없는 칸 비우기
- 적용 가능한 칸만 맞추기
- 문자열 내보내기 / 가져오기
- 현재 특성 바로 전환
- 비행 중 바를 `9번 바`로 함께 저장하고 적용
- 템플릿 정보에 특성명과 기록된 액션 개수 표시
- 메인 타이틀과 설정에서 현재 버전 확인 가능
- 퀘스트 후보 새로고침 / 안전 정리 / 전체 퀘스트 포기
- 전문기술 주간 체크 탭
  - profession별 주간 / 1회성 획득원을 숨은 퀘스트 기반으로 자동 추적
  - profession별 포인트 합계 표시
  - 전문기술 논문 / 주간 드랍 / 1회성 보물 / 평판 보상 / 일부 풍요 보상 / 처음 발견 보너스 자동 계산
- 전문기술 체크 오버레이
  - profession 아이콘 포함
  - 상세 / 요약 / 최소 3단 표시 모드
  - `XS / S / M / L / XL` 5단계 크기 조절
  - 드래그 이동 / 위치 저장
  - TomTom 설치 시 row 우클릭으로 다음 미완료 1회성 보물 waypoint 설정
- 한밤(Midnight) 지도 오버레이
  - 실버문 도심 주요 시설, 포탈 이름, 던전 / 구렁 / 공격대 입구, 평판 상인 위치를 읽기 쉬운 글자로 표시
  - Midnight 공식 던전 / 구렁 / 공격대 분류를 다시 검증해 잘못된 라벨 섞임과 map 오배치를 줄이도록 정리
  - 지도 확대 / 축소 상태를 읽어 축소 시에도 너무 작아지지 않게 보정
  - `던전` / `구렁` / `공격대` 접두어 줄과 이름 줄을 분리해 표시
  - 설정 탭에서 `주요 시설 / 포탈 / 전문기술 / 던전·공격대 / 구렁` 라벨을 각각 켜고 끌 수 있음
  - 지도 전환 / 확대·축소 / refresh 중 예외를 보수적으로 막는 안전장치 추가
- 캐릭터 스탯 오버레이
  - 상단에 `캐릭터 직업 - 특성(아이템레벨)` 헤더 표시
  - 치명 / 가속 / 특화 / 유연을 `평점 (퍼센트)` 형식으로 표시
  - 현재 특성의 PvE 일반 스탯 우선순위 표시
  - 탱커 특성은 회피 / 무막 / 막기를 함께 표시
  - 배경 없이 글자만 표시되며 좁은 힛트박스로 드래그 이동 가능
  - 값 영역 마우스 오버 시 캐릭터창에 가까운 스탯 설명과 DR 구간 안내 툴팁 표시
  - 설정 탭에서 `XS / S / M / L / XL` 5단계 크기 조절 가능

## 기본 사용법

1. 게임 안에서 `/abpm` 입력
2. `현재 접속 캐릭터` 탭에서 템플릿 이름 입력 후 `템플릿 저장`
3. 저장된 템플릿 선택
4. `액션바` 탭에서 적용 범위 선택
5. 필요하면 먼저 비교
6. 적용 또는 동기화 실행

## 전문기술 탭

- 현재 캐릭터의 기본 profession을 자동으로 읽어 profession별 카드로 표시합니다.
- 탭 안에서 `전문기술 포인트 오버레이 표시`를 바로 켜고 끌 수 있습니다.
- 탭 안에서 `XS / S / M / L / XL` 버튼으로 전문기술 오버레이 크기를 바로 바꿀 수 있습니다.
- 각 row는 `현재 획득 수 / 최대 획득 수`와 `획득 포인트 / 최대 포인트`를 같이 보여줍니다.
- 완료된 획득원은 카드 안에서 초록색 강조와 `완료` 표기로 바로 구분됩니다.
- 기본적으로 아래 항목들을 자동 추적합니다.
  - 주간 퀘스트
  - 주간 필드 / 상자 드랍
  - 주간 채집 드랍
  - 주간 분해 드랍
  - 전문기술 논문
  - 1회성 profession 보물
  - 평판 지식서 / 일부 풍요 지식서
  - 일부 profession의 처음 발견 보너스
- 각 profession 카드 하단의 `재스캔` 버튼으로 완료 플래그를 다시 읽을 수 있습니다.
- 전문기술 오버레이 상세 모드에서는 마우스 오버로 주간/1회성 획득원과 objective 상태를 툴팁으로 확인할 수 있습니다.
- 오버레이 요약은 한국어 기준으로 더 촘촘하게 압축해 작은 폭에서도 읽기 쉽게 정리했습니다.
- TomTom이 설치되어 있으면 전문기술 오버레이 row를 우클릭해 다음 미완료 1회성 보물 waypoint를 바로 찍을 수 있습니다.

## 템플릿 탭

- 현재 캐릭터 정보와 직업/특성 아이콘 표시
- 현재 특성 전환 버튼 제공
- 템플릿 저장 / 복제 / 목록 새로고침 / 삭제 버튼이 상단 입력칸 옆에 배치됩니다.
- 템플릿 목록에서 선택 가능
- `이동` 버튼으로 이전 / 다음 템플릿 선택 가능
- 같은 이름으로 저장하면 덮어쓰기 확인창이 먼저 표시됩니다.
- 템플릿 정보에는 저장 캐릭터, 특성 ID, 특성명, 기록된 액션 수, 주문/매크로/아이템 수가 표시됩니다.
- 템플릿 작업 영역에서 아래 기능 사용 가능
  - 선택한 템플릿 적용
  - 전체 액션바 비우기
  - 직전 작업 되돌리기
  - 문자열 내보내기
  - 문자열 가져오기

## 액션바 탭

적용 범위는 아래 방식으로 선택할 수 있습니다.

- 전체
- 바 1개
- 여러 바
- 선택 바
- 칸 번호

동기화 작업은 아래 기능을 제공합니다.

- 비어 있는 칸만 채우기
- 템플릿에 없는 칸 비우기
- 다른 칸만 바꾸기
- 선택 범위를 그대로 맞추기
- 적용 가능한 칸만 맞추기
  - 템플릿에는 있지만 현재 캐릭터가 바로 쓸 수 없는 스킬/매크로/아이템은 건너뜁니다.
  - 그런 칸은 고스트를 새로 만들지 않고 현재 칸을 유지합니다.
- `전체 액션바 비우기`는 한 번 비운 뒤 남은 칸을 다시 점검해서 한 번 더 비웁니다.

액션바 탭에서도 `직전 작업 되돌리기` 버튼으로 최근 1회 작업을 복구할 수 있습니다.
동기화 버튼 위에 마우스를 올리면 설명이 나오고, 버튼을 누르면 현재 선택 범위를 기준으로 자세한 설명이 하단 요약창에 표시됩니다.

## 퀘스트 탭

- `퀘스트 후보 새로고침`
  - 현재 퀘스트 목록을 다시 읽어 정리 후보를 새로 계산합니다.
- `안전 정리 실행`
  - 포기 가능
  - 완료 아님
  - 보고 가능 아님
  - 진행도 없음
  인 퀘스트만 정리합니다.
- `전체 퀘스트 포기`
  - 현재 퀘스트 목록에서 포기 가능한 퀘스트를 모두 포기합니다.
  - 진행 중인 퀘스트도 포함될 수 있습니다.
- 숨김 퀘스트, 작업형 퀘스트, 현상금 계열 항목은 후보 목록에서 제외합니다.
- 하단 후보 목록은 `안전 정리 대상 / 남겨둘 퀘스트 / 전체 포기 대상`으로 나뉘어 표시됩니다.
- 진행 중 퀘스트는 가능한 경우 objective 진행도도 함께 보여줍니다.

## 주의 사항

- 전투 중에는 보호된 액션바를 바로 변경할 수 없습니다.
- 전투 중 실행한 작업은 대기열에 들어가고, 전투가 끝나면 자동으로 다시 시도합니다.
- 없는 주문, 아이템, 매크로는 고스트 표시로 남을 수 있습니다.
- 고스트 표시가 남은 칸은 아이콘을 끌어 빼서 해제할 수 있습니다.
- 고스트 표시 위에 다른 스킬, 아이템, 매크로를 올려 덮어쓸 수도 있습니다.
- 고스트 자동 재시도는 플레이어가 커서에 다른 액션을 들고 있을 때는 멈춰 수동 덮어쓰기를 방해하지 않습니다.
- 현재 바 모델은 `1~9번 바`까지만 지원하며, `9번 바`는 비행 중 바입니다.
- `10~12번` 특수 바는 현재 별도 매핑하지 않습니다.
- 매크로는 이름과 본문을 기준으로 더 엄격하게 확인합니다.
- 되돌리기는 현재 접속 중 가장 최근 1회 작업만 복구할 수 있습니다.
- profession 체크는 현재 `한밤(Midnight) 12.0.1` 기준 숨은 퀘스트 / 완료 플래그 기반 자동 추적 구조입니다.
- 제작 주문, catch-up 같은 일부 예외 획득원은 아직 별도 자동 집계하지 않습니다.
- 한밤(Midnight) 지도 오버레이는 `한밤(Midnight)` 주요 지도와 쿠엘다나스 섬 기준의 정적 좌표를 사용합니다.
- profession 툴팁 이름은 한국어 클라이언트에서 퀘스트명을 우선 사용하고, 일부는 내장 번역 규칙으로 보정합니다.
- 1회성 보물은 한국어명이 모호하면 고유 영어명으로 표시될 수 있습니다.
- TomTom waypoint 기능은 TomTom 설치가 필요합니다.

## 스탯 오버레이

- `설정` 탭에서 표시 여부를 켜고 끌 수 있습니다.
- `설정` 탭에서 `XS / S / M / L / XL` 버튼으로 전체 표시 크기를 바꿀 수 있습니다.
- 치명 / 가속 / 특화 / 유연은 `평점 (퍼센트)` 형식으로 표시됩니다.
- 탱커 특성은 회피 / 무막 / 막기처럼 현재 적용되는 방어 확률도 추가로 표시됩니다.
- 마지막 줄에는 현재 특성 기준 PvE 일반 우선순위가 짧게 표시됩니다.
- 오버레이는 배경 없이 글자만 보이며, 왼쪽 드래그로 위치를 옮길 수 있습니다.
- 상단 한 줄에 현재 `캐릭터 직업 - 특성(아이템레벨)`이 함께 표시됩니다.
- 치명 / 가속 / 특화 / 유연의 값 영역 위에 마우스를 올리면 캐릭터창에 가까운 스탯 설명과 현재 DR 구간 안내 툴팁이 표시됩니다.
- 퍼센트는 항상 소수 둘째 자리까지 표시되고, 괄호 열도 맞춰집니다.
- 드래그 가능한 영역은 텍스트 폭에 가깝게 줄여 월드 클릭 방해를 줄였습니다.
- 2차 스탯은 rating 기준 `30 / 39 / 47 / 54 / 66%` 구간에 따라 퍼센트 숫자만 단계적으로 색이 바뀝니다.

## 미니맵 버튼

- 왼쪽 클릭: 애드온 창 열기 / 닫기
- 오른쪽 드래그: 버튼 위치 이동
- 축소된 사각 `AB` 버튼으로 표시됩니다.

## 추가 오버레이

- `설정` 탭에서 `전문기술 체크 오버레이`를 켜면 profession별 포인트 요약과 주요 획득원 진행도를 오버레이로 표시합니다.
- 오버레이는 왼쪽 드래그로 이동할 수 있고 위치가 저장됩니다.
- 오버레이는 상세 / 요약 / 최소 상태를 순환하며 전환할 수 있습니다.
- 전문기술과 스탯 오버레이는 각각 `XS / S / M / L / XL` 5단계 크기 조절을 지원합니다.
- `한밤(Midnight) 지도 오버레이`를 켜면 실버문, Eversong, Harandar, Voidstorm, Zul'Aman, 쿠엘다나스 섬 지도에서 주요 시설과 입구 이름을 글자로 표시합니다.
- 지도 확대 / 축소에 따라 글자 크기가 자동 조정되며, 축소 상태에서 외부 지역 라벨이 너무 작아지지 않도록 보정했습니다.
- 와우 `설정 > 애드온 > ABProfileManager` 아래에 템플릿 / 액션바 / 전문기술 / 퀘스트 하위 카테고리가 추가됩니다.
