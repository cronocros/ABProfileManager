# ABProfileManager v1.7.7 -> v1.10.0 대규모 업데이트 안내

배포 기준일: `2026-05-31`

`v1.7.7` 기준 이후 BIS 추천 장비 기능을 중심으로 큰 폭의 업데이트가 진행되었습니다. 이번 공지는 `v1.8.0`, `v1.9.0`, `v1.10.0`에 반영된 누적 변경 사항을 한 번에 정리합니다.

최신 버전: `v1.10.0`
직접 다운로드: `https://github.com/cronocros/ABProfileManager/releases/download/v1.10.0/ABProfileManager-v1.10.0.zip`

## 가장 크게 달라진 점

### 1. 한밤 시즌 1 BIS 추천 데이터 전면 갱신

BIS 추천 장비 카탈로그를 한밤 시즌 1 최신 오프라인 입력 기준으로 다시 생성했습니다.

- 전 클래스 `40개 전문화`의 쐐기와 티어 후보를 최신 데이터로 갱신했습니다.
- 기존 카탈로그에 있던 레이드 `285`행과 제작 `91`행은 유실되지 않도록 그대로 보존했습니다.
- 현재 BIS 카탈로그는 총 `3130`행입니다.
  - 쐐기: `2554`
  - 레이드: `285`
  - 제작: `91`
  - 티어: `200`
- 게임 실행 중에는 `Data/BISCatalog.lua` 하나만 읽습니다. 웹 조회, 런타임 병합, 데이터 정규화는 하지 않습니다.

쐐기 아이템은 같은 `itemID`라도 획득 경로에 따라 트랙과 능력치가 달라질 수 있습니다. 따라서 카탈로그가 정적 링크나 bonusID를 추측해서 확정하지 않도록 정책을 바꿨습니다.

- 던전 종료 후보: `영웅 3/6`, 아이템 레벨 `266`
- 위대한 금고·Voidcore 후보: `신화 1/6`, 아이템 레벨 `272`
- `itemID`만으로 최종 BIS, 영웅/신화 트랙, 최종 능력치를 확정하지 않습니다.
- 최종 선택은 실제 아이템 링크와 심크 결과를 함께 확인하는 것을 권장합니다.

### 2. BIS 오버레이를 검증 중심 목록으로 리디자인

BIS 오버레이는 단순 추천 목록에서, 출처와 검증 상태를 함께 확인하는 compact list 형태로 정리했습니다.

- 상단에서 전문화와 `쐐기 / 레이드 / 제작 / 티어` 출처 필터를 선택할 수 있습니다.
- 현재 전문화의 스탯 우선순위와 검증 상태를 헤더에 표시합니다.
- 각 행에서 아이템 아이콘, 이름, 부위, 출처, 보상 트랙, 검증 상태, 필터 후 순위를 확인할 수 있습니다.
- 필터를 바꾸면 현재 보이는 목록을 기준으로 `1순위 / 2순위 / 3순위+`를 다시 계산합니다.
- 아이템 툴팁에는 Base ItemID, 보상 프로필, 런타임 링크 필요 여부, 신화 트랙 후보/검증 상태, 심크 필요 안내를 구분해서 표시합니다.

안전성과 기존 동작도 유지했습니다.

- `GameTooltip:SetHyperlink()`를 직접 호출하지 않습니다.
- 쐐기/레이드 항목만 가능한 경우 모험 안내서 전리품 탭으로 이동합니다.
- 제작/티어 항목은 모험 안내서 랜딩 대상이 아닙니다.
- 아이템 정보 수신 이벤트는 현재 보이는 행만 다시 갱신합니다.

### 3. 즐겨찾기와 보유 장비 관리 추가

BIS 목록을 실제 파밍 체크리스트로 사용할 수 있도록 상태 저장 기능을 추가했습니다.

- 각 아이템 아이콘 앞에 `즐겨찾기`와 `보유` 체크박스가 표시됩니다.
- 즐겨찾기 아이템은 `무기` 위의 최상단 `즐겨찾기` 섹션으로 이동합니다.
- 보유한 아이템은 이름 위에 취소선을 표시합니다.
- 즐겨찾기와 보유 상태는 `캐릭터별·전문화별`로 저장됩니다.
- 재접속하거나 `/reload`를 실행해도 체크 상태를 유지합니다.
- 체크 표시는 작은 문자 대신 Blizzard 기본 체크 텍스처를 사용하도록 개선했습니다.
- 보유 취소선은 아이템명보다 앞 레이어에서 선명하게 표시합니다.

### 4. 쐐기 아이템 미리보기 개선

쐐기 BIS 항목에 마우스를 올리면 모험 안내서의 신화 던전 기준 미리보기를 사용합니다.

- Encounter Journal 신화 던전(M0) 기준
- `챔피언 1/6`
- 아이템 레벨 `246`

정적 링크를 임의로 만들지 않으면서도, 실제 던전 도감에서 확인 가능한 아이템 정보를 보여주기 위한 기준입니다.

### 5. 40개 전문화 스탯 우선순위 갱신

스탯 오버레이와 스탯 우선순위 표도 함께 갱신했습니다.

- 전 클래스 `40개 전문화`에 단일 대표 스탯 우선순위를 반영했습니다.
- BIS 정책 메타에도 같은 우선순위를 연결했습니다.
- 기존의 숨김 M+ 전용 우선순위 분기는 제거했습니다.
- 이전 SavedVariables 호환 키는 유지하므로 별도 설정 초기화는 필요하지 않습니다.
- 애드온 언어가 영어일 때 스탯 우선순위 표도 영문으로 표시합니다.
- 한국어 BIS 보상 트랙 표기는 `영웅 / 신화`로 현지화했습니다.

## 업데이트 후 확인 권장

- BIS 오버레이 열기/닫기와 전문화 전환
- `쐐기 / 레이드 / 제작 / 티어` 필터 조합

- 애드온 폴더를 최신 버전으로 교체한 뒤 `/reload`하거나 게임에 다시 접속하면 됩니다.
