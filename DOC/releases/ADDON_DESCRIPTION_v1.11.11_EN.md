# ABProfileManager

ABProfileManager is an all-in-one in-game utility addon for action bars, profession tracking, Midnight map guidance, BIS references, drop item levels, character stats, combat text settings, and Mythic+ record overlays.

It is designed to reduce setup time, weekly checklist fatigue, and external-note dependency by keeping progression information visible inside WoW.

## Main Features

- Save, compare, restore, partially apply, import, and export action-bar templates.
- Track weekly and one-time profession knowledge sources with compact overlay modes.
- Show Midnight map labels for portals, facilities, profession hubs, vendors, dungeons, and delves.
- Browse BIS recommendations by slot with `Mythic+ / Raid / Crafted / Tier` filters.
- Track favorite and owned BIS items per character and specialization.
- View verified Mythic+ `Myth 1/6 272` item tooltip snapshots when available.
- Use seasonal raid/tier/crafted preview links when validated, with normal Blizzard item tooltip fallback.
- Check drop item levels, Great Vault rewards, crests, and key resources in one overlay.
- See Mythic+ season-best score and dungeon names directly in Group Finder.
- Keep combat stats and current spec stat priorities visible during play.
- Manage Midnight combat text modes from addon settings.
