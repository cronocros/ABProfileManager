# ABProfileManager v1.5.6

릴리스 날짜: 2026-03-30

## 요약

이번 릴리스는 인게임 QA 후속 이슈를 반영한 BIS 데이터/툴팁 정리와 파티찾기 시즌 최고기록 오버레이 추가 릴리스입니다. 제작/레이드 필터가 실제 데이터까지 동작하도록 BIS 시즌 데이터 소스를 다시 구성했고, 오래된 던전 아이템도 가능한 한 현재 시즌 preview 툴팁으로 연결되게 보강했습니다.

## 주요 변경

### 1. BIS 인던 드랍 정보

- `Data/BISData_Method.lua`를 Method.gg current overall BIS 기준으로 교체했습니다.
- 기존 쐐기 리스트는 각 부위 `대체재 / 2순위`로 병합해, `쐐기 / 레이드 / 제작` 필터가 실제 데이터까지 반영되도록 정리했습니다.
- 리스트 컬럼을 `아이템명 / 출처 / BIS 여부 / 타입` 구조로 재배치했습니다.
- 오래된 던전 아이템은 Encounter Journal 시즌 preview loot를 `itemID` 우선, `아이템명` fallback 순으로 다시 매칭합니다.
- 아이템 클릭 시 가능하면 `Encounter Journal -> Dungeon -> Loot` 화면까지 더 확실히 랜딩하도록 delayed loot focus 경로를 추가했습니다.

### 2. 드랍템 레벨 정보

- 종합 탭 상단 공통 헤더를 제거하고 섹션별 요약만 남겼습니다.
- 구렁 탭에 `풍요 열쇠 사용` 보상 행을 추가해 영웅 `1/6` 보상을 별도 표기합니다.
- 우측 `나의 열쇠` 패널을 `오늘의 풍요 4개 + 열쇠 파편 + 복원된 열쇠` 구조로 재배치했습니다.
- `나의 문장`과 `위대한 금고` 사이 간격, 각 열 폭, 우측 패널 시작 위치를 다시 조정했습니다.
- 레이드 탭 월드보스 보상은 `?/?` 대신 정확한 등급 문자열로 표시합니다.

### 3. 파티찾기 시즌 최고기록

- 신화+ 던전 탭의 시즌 최고기록 던전 아이콘 위에 `단수 / 평점 / 최고기록 시간`을 직접 표시하는 `MythicPlusRecordOverlay`를 추가했습니다.
- Utility 탭에서 체크박스로 켜고 끌 수 있습니다.

### 4. 설정 UI

- WoW 기본 설정의 전투메시지 섹션 문구와 모드 버튼 폭을 더 줄여 overflow를 완화했습니다.

## 확인 메모

- 일부 구던전 BIS 아이템은 Encounter Journal 시즌 preview link를 받지 못하면 원본 아이템 툴팁으로 fallback 할 수 있습니다.
- `열쇠 파편`은 Blizzard 공개 API 기준으로 안전한 itemID를 아직 확정하지 못해 `-` fallback이 남아 있습니다.
- `오늘의 풍요 구렁` 4개 표시는 active delve / map POI 기준 best-effort 표시를 사용합니다.
