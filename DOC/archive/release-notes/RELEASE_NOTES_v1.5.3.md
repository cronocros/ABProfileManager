# ABProfileManager v1.5.3

릴리스 날짜: `2026-03-30`

## 요약

드랍템 레벨 오버레이와 BIS 오버레이를 인게임 피드백 기준으로 다시 정리한 릴리스입니다.

- ItemLevelOverlay: `나의 문장`을 반복 컬럼이 아닌 우측 통합 패널로 재구성
- ItemLevelOverlay: 쐐기 타이틀에 챔피언 / 영웅 / 신화 최고 강화 레벨 요약 추가
- BISOverlay: 던전/보스 기준 표시를 부위별 BIS / 대체 / 3순 구조로 재편
- BISOverlay: 베이스 아이템 툴팁 보정 대신 한밤 시즌 1 던전 트랙 요약 툴팁 적용
- Method.gg 기반 시즌 1 던전 BIS 보강 데이터 추가
- 영어 로케일의 BIS 슬롯 / 던전 / 툴팁 문구 정리

## 상세 변경

### 1. 드랍템 레벨 오버레이 재구성 (`UI/ItemLevelOverlay.lua`)

- 표 레이아웃을 `단 / 클리어보상 / 드랍문장 / 위대한 금고` 4열로 정리
- 우측 고정 패널에 현재 보유 문장을 한 번만 모아서 표시
- `위대한 금고` 열 폭을 늘리고 행별 rank 표기를 제거해 잘림을 줄임
- 쐐기 섹션 헤더에 `챔피언 ~263 / 영웅 ~276 / 신화 ~289` 요약 추가
- 탭 폭을 줄여 전체 레이아웃 밀도를 조정

### 2. BIS 오버레이 구조 변경 (`UI/BISOverlay.lua`)

- 던전 → 보스 → 아이템 구조를 부위 → 아이템 구조로 변경
- 각 부위 안에서 `BIS / 대체 / 3순` 배지로 우선순위를 정리
- 아이템 행 클릭 시 해당 던전을 모험 안내서에서 열 수 있도록 유지
- 툴팁은 베이스 아이템의 과거 품질/등급을 그대로 보여주지 않고,
  한밤 시즌 1 기준 `부위 / 출처 / 우선순위 / 클리어 보상 / 위대한 금고` 요약을 표시

### 3. BIS 데이터 보강 (`Data/BISData_Method.lua`)

- Method.gg Midnight Season 1 가이드의 Mythic+/Overall 표에서
  시즌 1 던전 드랍만 추출한 보강 데이터를 추가
- 기존 `Data/BISData.lua` 수기 데이터와 itemID 기준으로 병합
- 총 34개 스펙에 대해 던전 BIS 부위 데이터를 보강

### 4. 로케일 및 문서 갱신

- `Locale_Additions.lua`에 BIS 슬롯 / 던전 / 배지 / 툴팁 키 추가
- `README`, `ADDON_INTRO`, `ARCHITECTURE`, `HANDOFF`, `CLAUDE`를 `v1.5.3` 기준으로 갱신
- 최신 릴리스 노트를 `RELEASE_NOTES_v1.5.3.md`로 교체

## 참고

- 로컬 패키지: `dist/ABProfileManager-v1.5.3.zip`
- Midnight 신규 던전 EJ ID는 여전히 일부 미확인 상태라 클릭 시 모험 안내서만 열리고 해당 던전으로 바로 이동하지 않을 수 있습니다.
- 문장 통화 ID는 영웅(3345) 외 일부가 추정값이라 인게임 검증이 남아 있습니다.
