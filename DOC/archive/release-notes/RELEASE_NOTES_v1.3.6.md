# ABProfileManager v1.3.6

배포일: `2026-03-12`

## 핵심 변경

- 한밤(Midnight) 지도 오버레이의 map 해석을 더 보수적으로 바꿔, 다른 지역의 던전/구렁 라벨이 잘못 섞여 보일 가능성을 줄였습니다.
- 던전 / 구렁 / 공격대 라벨은 `접두어 줄 + 이름 줄` 구조로 통일했고, 한국어 길이 기준 줄바꿈 규칙을 다시 정리했습니다.
- 실버문 전문기술 라벨과 외부 지역 라벨 크기를 다시 미세 조정하고, 지원 지도 이름 alias를 보강했습니다.
- 전문기술 탭 카드의 우측 값 영역을 더 압축해 `0/2` 같은 포인트 값이 카드 밖으로 넘치지 않도록 보정했습니다.
- 전문기술 오버레이는 상세 모드 정렬을 정리했고, 마우스 오버 툴팁으로 주간/1회성 획득원과 objective 상태를 확인할 수 있게 했습니다.
- 퀘스트 후보 목록은 섹션 제목 위치와 항목 간 줄 간격을 다시 정리해 읽기 쉽게 조정했습니다.
- 설정 탭 세션 요약에 저장 범위를 명시하고, 지도 라벨 체크박스 영역 배치를 더 촘촘하게 정리했습니다.
