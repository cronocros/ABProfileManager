# ABProfileManager v1.3.7

이번 패치는 전문기술 카드 가독성과 한밤(Midnight) 지도 라벨 예외 정리에 집중한 안정화 릴리스입니다.

## 핵심 변경

- 전문기술 탭 카드의 우측 포인트 영역을 `포인트 / 수치` 2줄 블록으로 재배치해 `0/2` 값이 카드 밖으로 넘치지 않도록 수정
- 완료된 전문기술 획득원은 초록 강조와 `완료` 표기로 바로 구분되도록 보강
- 전문기술 오버레이 요약은 한국어 기준으로 더 촘촘하게 압축하고, 전체 완료 profession은 색상으로 구분
- 설정 탭에서 `전문기술 체크 오버레이` 토글을 왼쪽 기본 설정 컬럼으로 이동해 지도 라벨 체크박스 영역 overflow를 완화
- 제작자 정보 표기를 `이름` + `(이메일)` 2줄 형식으로 정리
- 한밤(Midnight) 지도 라벨은 `구렁 허브`, `살육의 거리`, `교정의 재앙` 같은 핵심 예외를 직접 줄바꿈 지정
- 지도 라벨은 충돌이 적을 때는 POI 아이콘을 가리지 않는 선에서 더 가까운 위치에 놓이도록 후보 오프셋을 다시 조정
- 퀘스트 탭 `안전 정리 대상` 목록에도 빈 줄을 넣어 세 목록의 간격 규칙을 통일
