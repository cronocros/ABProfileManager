# ABProfileManager v1.3.4

배포일: `2026-03-12`

## 핵심 변경

- 한밤(Midnight) 지도 오버레이를 규칙 기반 라벨 시스템으로 정리했습니다.
  - `던전:` / `구렁:` / `공격대:` 접두어 적용
  - 라벨 줄바꿈과 겹침 완화 보강
  - 축소 상태에서도 지나치게 작아지지 않도록 보정
  - 평판 / 풍요 상인은 NPC 이름 대신 기능명 중심으로 정리
- 전문기술 오버레이와 스탯 오버레이 크기 조절을 `XS / S / M / L / XL` 5단계 프리셋으로 확장했습니다.
- 템플릿 탭 현재 캐릭터 정보에 `특성 ID + 특성명`을 함께 표시하도록 보강했습니다.
- 퀘스트 후보 목록을 `안전 정리 → 남겨둘 퀘스트 → 전체 포기` 순서로 재정리하고, 진행 중 퀘스트 objective 진행도 요약을 표시하도록 개선했습니다.
- 설정 탭 현재 세션 요약을 스크롤 가능한 박스로 바꾸고, 제작자 이메일 `crono1232@gmail.com`을 함께 표시합니다.
- 스탯 오버레이는 버프와 능력치 변화에 더 안정적으로 반응하도록 관련 이벤트를 보강했습니다.
