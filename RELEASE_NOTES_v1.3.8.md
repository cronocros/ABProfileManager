# ABProfileManager v1.3.8

이번 패치는 한밤(Midnight) 지도 데이터 정합성과 전문기술 UI 가독성 회귀를 함께 정리한 안정화 릴리스입니다.

## 핵심 변경

- 한밤(Midnight) 지도 오버레이의 던전 / 구렁 / 공격대 목록을 공식 분류 기준으로 다시 확인하고, 잘못된 카테고리와 누락된 입구 라벨을 정리
- 이미 줄바꿈된 한국어 지도 라벨은 다시 자동 분절되지 않도록 로직을 수정해 `전문기술 / 허브`, `보이드스톰 / 포탈` 같은 수동 줄바꿈이 그대로 유지되게 보강
- 실버문 전문기술 허브, 스톰윈드 포탈, 쿠엘다나스 / 하란다르 / 보이드스톰 / 줄아만의 주요 던전·구렁·공격대 라벨 위치를 다시 조정
- 전문기술 탭 카드 헤더는 아이콘을 오른쪽으로 옮기고, 우측 포인트 영역을 작은 2줄 블록으로 다시 구성해 `0/2` 값이 카드 밖으로 넘치지 않도록 수정
- 전문기술 오버레이 상세 모드는 `주간 | ...`, `1회성 | ...` 형식의 정렬된 2열 레이아웃으로 다시 정리
- 전문기술 오버레이 툴팁은 `완료 / 진행` 상태를 분명하게 표시하고, 한국어 제목을 찾지 못한 objective는 한국어 fallback 이름으로 대체
- 퀘스트 후보 목록은 제목 / 퀘스트 ID / 진행도 / 유지 사유를 색으로 분리해 더 읽기 쉽게 정리
- 설정 세션 요약의 제작자 정보는 `제작자 / 이름(이메일)` 한 줄 형식으로 정리
