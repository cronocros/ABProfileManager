# ABProfileManager v1.3.2

패치 릴리스입니다.

## 이번 버전 핵심

- 한밤(Midnight) 지도 오버레이 라벨 규칙을 다시 정리했습니다.
  - 던전 / 구렁 / 주요 시설 라벨의 과한 공백을 줄이고, 긴 이름은 더 읽기 쉬운 두 줄 표기로 정리했습니다.
  - 지도 확대 / 축소에 맞춘 글자 크기 조정은 유지하되, 기본 크기와 폭을 다시 다듬어 긴 이름이 과하게 퍼지지 않도록 보정했습니다.
  - 쿠엘다나스 섬 지도에 `마법학자의 정원`과 `태양샘 고원` 입구 라벨을 추가했습니다.
- 전문기술 탭 카드 레이아웃을 다시 정리했습니다.
  - 오버레이 체크박스 추가 후 카드가 하단 영역을 덮던 문제를 완화했습니다.
  - 카드 설명과 section 제목, 진행 표기를 더 짧고 읽기 쉽게 정리했습니다.
  - `Treatise` 표기는 `전문기술 논문`으로 통일했습니다.
- 퀘스트 탭의 정리 기준 설명을 일반 사용자가 더 바로 이해할 수 있는 문구로 정리했습니다.
- 설정과 전문기술 탭의 `Midnight` 표기를 `한밤(Midnight)` 기준으로 맞췄습니다.

## 참고

- 지도 라벨은 정적 좌표 기반이라, 실제 게임 패치 후 좌표가 달라지면 미세 조정이 추가로 필요할 수 있습니다.
- 전문기술 자동 추적은 여전히 숨은 퀘스트 / 완료 플래그 기반이며, 제작 주문과 catch-up 같은 일부 예외 획득원은 별도 자동 집계하지 않습니다.
