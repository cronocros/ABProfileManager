# ABProfileManager v1.1.0 Release Notes

릴리스 날짜:
- 2026-03-12

## 핵심 추가 기능

### 1. 전문기술 주간 체크 탭

- 메인 창에 `전문기술` 탭이 추가되었습니다.
- 현재 캐릭터의 기본 전문기술을 자동으로 읽어 profession별 카드로 표시합니다.
- 각 profession마다 아래 항목을 수동 체크할 수 있습니다.
  - 주간 퀘스트 / 주간 상자·필드 드랍 / Treatise
  - 채집 전문기술의 주간 채집 드랍
  - 마법부여의 주간 분해 드랍
  - 1회성 전문기술 보물
  - 일부 profession의 처음 발견 보너스
- 행마다 `현재 획득 수 / 최대 획득 수`와 `획득 KP / 최대 KP`를 함께 표시합니다.
- 주간 항목은 캐릭터별로 저장되며, 주간 리셋 키가 바뀌면 자동으로 초기화됩니다.

### 2. 전문기술 체크 오버레이

- 설정에서 켜고 끌 수 있는 소형 오버레이가 추가되었습니다.
- profession별 `주간 KP / 1회성 KP` 요약만 간단하게 표시합니다.
- 드래그 앤 드롭으로 위치를 옮길 수 있고 위치가 저장됩니다.

### 3. 실버문 지도 오버레이

- 설정에서 `실버문 지도 오버레이`를 켜면 Midnight 실버문 월드맵에 주요 위치가 글자로 표시됩니다.
- 현재 표시 대상:
  - 은행 / 금고
  - 여관
  - 경매장
  - 도시 포탈
  - M+ 포탈
  - 전문기술 허브
  - 제작 주문
  - 정복 / 명예 허브
  - 창조 촉매
  - 탐험 허브
  - 주요 전문기술 상인 위치

## 기타
