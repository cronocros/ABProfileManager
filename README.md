## 주요 기능

- 현재 액션바 배치를 템플릿으로 저장
- 같은 이름으로 저장 시 덮어쓰기 확인
- 선택한 템플릿 복제
- 저장한 템플릿 선택, 적용, 삭제
- 템플릿 저장/복제/목록 새로고침 옆 상단 행에 `템플릿 삭제` 배치
- 템플릿 문자열 내보내기 / 가져오기
- 템플릿 탭에서 현재 특성 바로 전환
- 템플릿 탭에서 전체 액션바 비우기
  - 1차 비우기 후 남은 칸을 한 번 더 점검해서 재비우기
- 최근 1회 작업 되돌리기
- 적용 범위 선택
  - 전체 액션바
  - 바 1개
  - 여러 바
  - 원하는 바 번호 직접 선택
  - 칸 번호 직접 입력
- 현재 액션바와 템플릿 비교
- 비교 결과 전체를 스크롤로 확인
- 동기화 작업
  - 비어 있는 칸만 채우기
  - 템플릿에 없는 칸 비우기
  - 다른 칸만 바꾸기
  - 현재 적용 가능한 칸만 맞추기
  - 선택 범위를 템플릿 그대로 맞추기
  - 버튼 hover 설명과 클릭 시 요약 설명 제공
- 비행 중 페이지 전환 바를 `9번 바`로 함께 저장/비교/적용
- 메인 타이틀과 설정 탭에 현재 버전 표시
- 템플릿 정보에 특성명, 기록된 액션 수, 주문/매크로/아이템 수 표시
- 퀘스트 정리 탭
  - 퀘스트 후보 새로고침
  - 안전 정리 실행
  - 전체 퀘스트 포기
- 전투 중 작업 대기열 처리
- 없는 주문/아이템/매크로는 고스트 오버레이 표시
- 고스트 오버레이는 드래그로 해제하거나 다른 액션을 올려 덮어쓸 수 있음
- 미니맵 버튼은 축소된 사각 `AB` 버튼형으로 표시
- 캐릭터 스탯 오버레이 표시
  - 치명 / 가속 / 특화 / 유연을 `평점(퍼센트)` 형식으로 표시
  - 현재 특성의 PvE 일반 스탯 우선순위 표시
  - 탱커 특성은 회피 / 무막 / 막기 같은 방어 스탯을 추가 표시
  - 배경 없는 글자형 레이아웃, 드래그 이동, 위치 저장 지원
  - 값 영역 마우스 오버 시 스탯 설명과 DR 구간 안내 툴팁 표시
- 한국어 기본 UI, 영어 옵션 제공
- 미니맵 버튼 제공
- 미니맵 버튼은 단순 버튼형 UI 사용
- 디버그 로그는 현재 접속에만 유지

## 매크로 처리

- 매크로 슬롯도 템플릿에 저장됩니다.
- 적용 시 이름과 본문을 기준으로 더 엄격하게 검증합니다.
- 같은 이름이지만 본문이 다르면 잘못 적용하지 않습니다.
- 같은 이름의 매크로가 여러 개라 정확히 구분되지 않으면 건너뜁니다.

## UI 구성

- 메인 창 상단 제목:
  - `액션바매니저 (ABProfileManager) 제작자 : 밍밍이와코코  v1.0.5`
- `현재 접속 캐릭터` 탭:
  - 현재 캐릭터 정보
  - 직업 / 특성 아이콘
  - 특성 전환 버튼
  - 템플릿 저장 / 복제 / 목록 새로고침 / 삭제 상단 행
  - 템플릿 목록
  - 이전 / 다음 템플릿 이동 버튼
  - 템플릿 정보에 특성명과 기록 통계 포함
  - 템플릿 작업 버튼 세로 배치
- `액션바` 탭:
  - 템플릿 정보
  - 비교 결과 전체 스크롤 표시
  - 적용 범위 선택
  - 동기화 작업
  - 직전 작업 되돌리기
- `설정` 탭:
  - 언어
  - 미니맵 버튼 표시
  - 확인창 사용 여부
  - 디버그 로그
  - 스탯 오버레이 표시 여부
- `퀘스트` 탭:
  - 정리 기준 요약
  - 안전 정리 대상 / 전체 포기 대상 / 유지 대상 목록
  - 안전 정리 실행
  - 전체 퀘스트 포기

## 슬래시 명령

- `/abpm`
- `/abpm help`
- `/abpm list`
- `/abpm savetemplate 내 템플릿`
- `/abpm apply template 내 템플릿`
- `/abpm apply template "공용 레이아웃" bar 2`
- `/abpm apply template 내 템플릿 bars 2 4 clear`
- `/abpm apply template 내 템플릿 slots 1 48 noclear`
- `/abpm delete template "공용 레이아웃"`
- `/abpm undo`
- `/abpm debug on`
- `/abpm debug off`
- `/abpm debug status`
- `/abpm clear 1 196`

템플릿 이름에 `full`, `bar`, `bars`, `slots` 같은 예약어가 들어가면 따옴표로 감싸는 편이 안전합니다.

## 현재 제약

- 실제 변경 허용 범위는 현재 `1-132`, `145-180`을 사용합니다.
- 전체 저장/선택 모델은 `1-196` 기준으로 동작합니다.
- 현재 바 모델은 `1~9번 바`까지만 지원합니다.
- `9번 바`는 비행 중 페이지 전환 바입니다.
- `10~12번` 특수 바는 현재 별도 매핑하지 않습니다.
- 되돌리기는 현재 세션 기준 최근 1회만 지원합니다.
- 스탯 오버레이의 유연 표시는 현재 캐릭터 총 유연 보너스 기준으로 보여줍니다.
- 2차 스탯은 rating으로 얻는 수치가 `30 / 39 / 47 / 54 / 66%` 구간에 들어가면 퍼센트 숫자만 단계적으로 색이 변합니다.
- 퀘스트 안전 정리는 포기 가능 + 완료 아님 + 보고 가능 아님 + 진행도 없음 기준만 정리합니다.
- 전체 퀘스트 포기는 현재 목록에서 포기 가능한 퀘스트만 대상으로 합니다.
- 퀘스트 탭은 일반 퀘스트 기준으로 동작하며, 숨김/작업/현상금 계열 항목은 후보에서 제외합니다.
- 하단 요약창의 오른쪽 끝 정렬은 사용자 요청으로 후순위 보류 상태입니다.
- 이 작업 환경에는 `lua`/`luac`가 없어 정적 문법 검사는 아직 실행하지 못했습니다.
- 메인 타이틀과 설정의 버전 표시는 `.toc` 메타데이터를 기준으로 동적으로 읽습니다.
- 배포 스크립트는 `dist/`에 릴리스 ZIP을 만들고, `backups/source/`에 소스 백업 ZIP도 함께 만듭니다.

## 보안 메모

- 동적 코드 실행 경로는 사용하지 않습니다.
- 문자열 가져오기는 코드 실행이 아니라 데이터 파싱 방식입니다.
- 가져오기 문자열은 길이/줄 수/중복 슬롯/지원 액션 종류를 검증합니다.
- `전체 퀘스트 포기`는 설정과 무관하게 항상 확인 모달을 거칩니다.
