# ABProfileManager v1.3.1

릴리스 날짜: `2026-03-12`

## 핵심 변경

- Midnight 지도 오버레이 글자 크기를 다시 낮춰 긴 던전 / 구렁 이름이 과하게 길어지지 않도록 정리
- 지도 확대 / 축소 상태에 따라 지도 오버레이 글자 크기를 완만하게 자동 조정하도록 보강
- 전문기술 탭에 `전문기술 포인트 오버레이 표시` 체크박스를 추가해 설정 탭과 같은 값을 바로 제어할 수 있게 변경
- profession 오버레이를 `상세 / 요약 / 최소` 3단 표시 모드로 확장
- 메인 창을 일반 Blizzard 창보다 앞에서 보이도록 `Raise / Toplevel / FrameStrata` 동작 보강
- 인트로 문서와 사용자 문서를 일반 WoW 플레이어 기준으로 다시 정리하고 소개 톤을 개선

## 참고

- 지도 오버레이 줌 연동은 고정 크기 점프가 아니라 완만한 크기 조정 방식으로 적용했습니다.
- profession 오버레이 최소 모드에서는 제목과 버튼만 남기고, 요약 모드에서는 profession별 핵심 진행도만 유지합니다.
