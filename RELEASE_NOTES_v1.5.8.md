# ABProfileManager v1.5.8

릴리스 날짜: 2026-03-31 (유지보수 재패키징)

## 유지보수 재패키징

- `MerchantHelper`, `MailHistory`, `WorldEventOverlay`, `WorldEventSchedule`는 현재 사용자 기능으로 노출되지 않는 비활성/미완성 상태이므로 TOC 로드 목록에서 제외했습니다.
- profession 추적은 완료 퀘스트 스냅샷이 실제로 바뀐 경우에만 요약 캐시와 overlay 계산을 무효화하도록 보수적으로 최적화했습니다.
- profession overlay tooltip은 refresh 시점이 아니라 hover 시점에만 생성되도록 바꿔 follow-up refresh 구간의 CPU/메모리 압박을 줄였습니다.
- 메인 창이 닫혀 있을 때는 내부 탭 패널 전체를 더 이상 매번 refresh하지 않고, 보이는 UI와 overlay만 갱신하도록 정리했습니다.

## 요약

이번 릴리스는 `BIS 인던 드랍 정보`와 `파티찾기 시즌 최고기록` 오버레이를 인게임 QA 기준으로 마무리하고, 루트/기술 문서를 현재 동작에 맞게 전면 갱신한 문서+QA 릴리스입니다.

## 주요 변경

### 1. BIS 인던 드랍 정보 오버레이 정리

- 부위별 `BIS / 대체 / 3순위` 구조를 유지하면서 `드랍 출처 / 유형 / BIS 여부` 열 배치를 다시 조정했습니다.
- `쐐기 / 레이드 / 제작` 필터는 체크박스형 버튼으로 바꿔 on/off 상태를 더 분명하게 표시합니다.
- 헤더에는 `참고용, 실제 템은 직접 확인` 안내를 추가해 BIS 표의 용도를 명확히 했습니다.
- 제작과 촉매 항목은 더 이상 Encounter Journal로 랜딩하지 않습니다.
- BIS 행 hover 툴팁은 시즌 preview 안정성 문제 때문에 비활성화했습니다.

### 2. BIS 랜딩 / 이름 매칭 / 깜빡임 보강

- 아이템 캐시가 늦게 들어와도 전체 리스트를 다시 그리지 않고, 현재 보이는 행만 갱신하도록 바꿔 깜빡임과 과도한 새로고침을 줄였습니다.
- `공결탑 제나스`는 실제 모험 안내서 이름과 tier 13 / instanceID 1314 기준으로 직접 보정했습니다.
- `알게타르 대학`은 공식 한글명과 instanceID 2526 기준으로 alias 및 랜딩 경로를 보강했습니다.
- 영어 source label과 한글 source label을 함께 정규화해 던전/레이드 분류와 모험 안내서 매칭을 더 안정적으로 만들었습니다.

### 3. 드랍템 레벨 / 우측 패널 문서 정리

- `ItemLevelOverlay`의 `나의 문장` / `나의 열쇠` 패널 구조와 현재 시즌 참조 범위를 문서에 반영했습니다.
- 로컬 패키지, 릴리스 노트, README 다운로드 링크를 `v1.5.8` 기준으로 정리했습니다.

### 4. 파티찾기 시즌 최고기록 오버레이 재정렬

- 시즌 최고기록 던전 아이콘에는 이제 `평점 + 던전명`만 표시합니다.
- 평점/던전명 위치를 아이콘 하단 기준으로 다시 정렬했습니다.
- `윈드러너 첨탑`, `삼두정의 권좌`, `공결탑 제나스`, `사론의 구덩이`, `마법학자의 정원`, `마이사라 동굴`, `알케타르 대학`은 강제 줄바꿈 규칙을 적용했습니다.
