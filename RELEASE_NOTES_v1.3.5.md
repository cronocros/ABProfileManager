# ABProfileManager v1.3.5

배포일: `2026-03-12`

## 핵심 변경

- 전문기술 탭 카드의 우측 진행 / 포인트 영역을 더 압축해 긴 항목이 카드 밖으로 넘치지 않도록 다시 정리했습니다.
- 실버문과 외부 지역 지도 라벨 크기를 한 단계 더 키우고, 특히 전문기술 허브 라벨이 더 잘 읽히도록 밀집도 보정을 완화했습니다.
- 한밤(Midnight) 지도 오버레이에 `주요 시설 / 포탈 / 전문기술 / 던전·공격대 / 구렁` 카테고리별 체크박스를 추가해 필요한 정보만 골라서 볼 수 있게 했습니다.
- 퀘스트 후보 목록은 `안전 정리 대상 / 남겨둘 퀘스트 / 전체 포기 대상` 3개 섹션으로 나눠서 읽기 쉽게 정리했습니다.
- 설정 탭 현재 세션 요약은 `세션 / 빠른 안내 / 제작자 정보` 문단으로 재구성하고, 기호는 `▶` / `→` 중심으로 정리했습니다.
- 인트로와 사용자 문서도 `v1.3.5` 기준으로 다시 정리해, 핵심 기능과 다운로드 경로를 더 빠르게 파악할 수 있게 했습니다.
