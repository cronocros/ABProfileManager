# ABProfileManager v1.4.1

릴리스 날짜: `2026-03-16`

## 요약

이번 릴리스는 profession overlay 정렬/tooltip 복구와 Midnight 지도 오버레이 보정을 중심으로 마감한 패치 버전입니다.

- profession tooltip 범례와 완료/미완료 색상 복구
- `주  간:` / `1회성:` 정렬 고정과 TomTom 우클릭 hitbox 확대
- Midnight 지도 좌표 보정, 중복 `평판상인` 정리, 큰 글자 충돌 완화
- 퀘스트 후보목록 기호 깨짐 제거와 휠 스크롤 보정
- 전투메시지 설정 UI를 표출 방식 전용으로 단순화하고 적용 retry 보강

## 상세 변경

### 1. Profession Overlay

- tooltip 상단 범례를 다시 노출하고, 완료/미완료 상태를 범례와 같은 색으로 본문에 적용했습니다.
- 상세 보기의 접두어는 한국어 기준 `주  간:` / `1회성:`으로 다시 고정해 line start가 흔들리지 않도록 정리했습니다.
- `1회성` tooltip은 보물과 처음발견보너스의 완료/미완료 목록을 더 이상 길게 나열하지 않습니다.
- 평판 지식서와 풍요 지식서는 계속 개별 항목 목록을 유지합니다.
- TomTom이 설치된 경우 `1회성` 줄 전체에서 우클릭이 잡히도록 hitbox를 넓혔습니다.

### 2. Midnight 지도 오버레이

- 줄아만에서 중복으로 보이던 `평판상인` 표시를 정리했습니다.
- 영원노래 숲, 하란다르, 보이드스톰 일부 포탈/구렁/시설 좌표를 최근 피드백 기준으로 다시 보정했습니다.
- 지도 글자 크기 상한을 더 넓히고, 큰 글자에서 라벨이 겹칠 때 더 멀리 피하도록 배치 후보와 충돌 점수를 조정했습니다.
- 지원하지 않는 child/detail map은 부모 지도 라벨을 따라오지 않도록 더 보수적으로 숨깁니다.

### 3. 퀘스트와 설정 패널

- 퀘스트 후보목록 앞에 쓰던 특수 기호를 ASCII로 바꿔 문자 깨짐을 없앴습니다.
- read-only scroll box도 마우스 휠 스크롤이 안정적으로 전달되도록 보강했습니다.
- 설정 탭은 `일반 설정`, `글자 크기`, `개요`, `전투메시지` 박스로 다시 나눴습니다.
- 전투메시지 설정은 이제 WoW 기본 on/off를 덮어쓰지 않고, `위로 / 아래로 / 부채꼴` 표출 방식과 방향성 분산만 관리합니다.
- 로그인/월드 진입 직후 짧은 재적용 retry를 추가해 `부채꼴` 모드 적용 누락 가능성을 더 줄였습니다.
