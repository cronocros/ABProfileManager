# ABProfileManager v1.3.10

## 핵심 변경

- 한밤(Midnight) 지도 오버레이가 던전 / 공격대 / 구렁 내부 지도에 잘못 표시되던 버그를 차단했습니다.
- 지도 라벨은 이미 줄바꿈된 한국어 텍스트를 다시 쪼개지 않도록 측정 로직을 다시 정리했습니다.
- 전문기술 오버레이와 퀘스트 후보 목록에서 색상 코드 조각이 노출되던 렌더링 오류를 수정했습니다.
- 전문기술 오버레이 상세 모드의 `주간 / 1회성` 열 정렬을 다시 맞추고 잘리던 접두어를 보정했습니다.
- TomTom 설치 시 전문기술 오버레이 row 우클릭으로 다음 미완료 1회성 보물 waypoint를 찍을 수 있습니다.

## 세부 변경

- profession treasure waypoint 데이터셋 추가
- `TomTomBridge` 모듈 추가
- 전문기술 1회성 보물 이름 fallback 개선
- 설정 세션 요약과 오버레이 툴팁에 TomTom 안내 문구 추가
